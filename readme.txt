Image Slideshow Wordpress plugin helps to display image slideshow in a wordpress website.

== Description ==

Image Slideshow Wordpress plugin helps to display image Slideshow in a wordpress website. If you want to display 10 images in a webpage as an image slideshow you have to go to that particular webpage and upload your images using Wordpress image upload function. You should then enter [pearl_slideshow_display] on the location of the webpage where you need the image slideshow to be displayed.It is very easy to use and simple to implement.

This plugin is very helpful for those who manage a portfolio site or any site displaying a series of images or photos.There is option to customise the display by modifying the dimesions, background colour, captions etc.



== Installation ==

1. Download and Unzip the file.
2. Upload the folder named `pearl_slideshow` to the `/wp-content/plugins/` directory.
3. Upload images to the webpage where you want to display image slideshow.
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Go to the webpage where you want to display the Image Slideshow and enter `[pearl_slideshow_display]`
